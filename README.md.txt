# AI Powered Image Colorizer

## Overview

AI Powered Image Colorizer is a Python application that converts black-and-white images into realistic color images using Deep Learning and OpenCV DNN.

## Features

* Upload grayscale images
* AI-powered image colorization
* OpenCV Deep Neural Network integration
* Streamlit web interface
* Download colorized images

## Technologies Used

* Python
* OpenCV
* NumPy
* Streamlit
* Deep Learning

## Project Structure

AI Powered Image Colorizer
├── app.py
├── colorizer.py
├── models
├── uploads
├── outputs
├── screenshots
└── README.md

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

python -m streamlit run app.py
